이런 프로시저가 있을 때:
```sql
  CREATE OR REPLACE PROCEDURE TEST_PROCEDURE_01(
    PARAMETER1 IN VARCHAR2,
    RESULT OUT VARCHAR2
  )
  AS
  BEGIN
    RESULT := 'BEGIN';
    DBMS_OUTPUT.PUT_LINE(PARAMETER1);
    RESULT := 'END';
  END;
  /
```
아래처럼 호출하면:
```sql
DECLARE
  RESULT VARCHAR2(100);
BEGIN
  TEST_PROCEDURE_BY_NORITER('hello!', RESULT);
  DBMS_OUTPUT.PUT_LINE(RESULT);
END;
COMMIT;
```
요로코롬 찍힌다:
```
  hello!
  END
```

끗.
