## 현재 시각

- 오라클: `SYSDATE`
- MySQL: `NOW()`

## 테이블 별칭

- 오라클: 테이블 별칭을 쌍따옴표로 감싸기 가능. `AS` 키워드 사용 불가. ```FROM SOME_TABLE "SOME_ALIAS"```
- MySQL: 테이블 별칭을 쌍따옴표로 감싸기 불가. `AS` 키워드 사용 가능. ```FROM SOME_TABLE AS SOME_ALIAS```, ```FROM SOME_TABLE AS `SOME_ALIAS````

## null 처리

- 오라클: `NVL( expr1, expr2 )`
- MySQL: `IFNULL( expr1, expr2 )`
