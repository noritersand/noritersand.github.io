## 환경 변수 추가

```bash
export LANG = "ko.KR.UTF-8"
```

## 파일 내용 읽어서 환경 변수로 설정하기

쉽죠잉

```
export VARIABLE_NAME = $( cat FILE_LOCATION )
```

```bash
export PID=$(cat /usr/local/tomcat8.5-6/bin/catalina.pid)
echo $PID
```
